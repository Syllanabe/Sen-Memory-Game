# Sen Memory Game - Phase 1

## Phase 1 : Game Board Setup

Cette phase consiste à mettre en place le plateau du jeu Memory Game.

## Objectifs de la phase

- Créer la structure HTML du jeu.
- Créer une grille de cartes.
- Préparer une face cachée et une face visible pour chaque carte.
- Mélanger les cartes de manière aléatoire.
- Ajouter un bouton pour recommencer la partie.
- Prévoir une condition de fin de partie.
- Afficher un message de félicitations lorsque toutes les paires sont trouvées.

## Fonctionnalités réalisées

- Plateau de jeu créé avec HTML.
- Cartes générées dynamiquement avec JavaScript.
- Design responsive avec CSS.
- Bouton Recommencer fonctionnel.
- Bouton de test de fin de partie pour vérifier la condition de victoire.

## Technologies utilisées

- HTML
- CSS
- JavaScript

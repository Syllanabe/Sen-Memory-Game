# Sen Memory Game - Phase 2

## Phase 2 : Card Flipping and Matching

Cette phase ajoute l'interactivité principale du jeu Memory Game.

## Objectif de la phase

L'objectif est de permettre au joueur de retourner les cartes et de vérifier si deux cartes sélectionnées forment une paire identique.

## Fonctionnalités réalisées

- Ajout d'événements de clic sur chaque carte.
- Retournement fluide des cartes avec une animation CSS.
- Sélection de deux cartes par le joueur.
- Comparaison des symboles des deux cartes retournées.
- Les cartes identiques restent visibles.
- Les cartes différentes se retournent automatiquement après un court délai.
- Blocage temporaire du plateau pendant la comparaison.
- Affichage du nombre de paires trouvées.
- Message de félicitations lorsque toutes les paires sont trouvées.
- Bouton Recommencer pour relancer une nouvelle partie.

## Technologies utilisées

- HTML
- CSS
- JavaScript
